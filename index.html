AI-Based Sensor for Orientation Tracking

Research Project: Developed an AI-powered IMU orientation sensor using neural networks (LSTM) to track and predict 3D orientation between two IMUs. Includes designing a 3DOF mechanism (ball joint) for testing predictions.

Technical Details: Uses two MPUs and a magnetometer, with 9 input features and 3 response outputs (J2, Y, Z axes). Implements sequential data splitting, tanh activation, min-max normalization, and 0.25 dropout to prevent overfitting.

Visualization: Green for training, yellow for validation, red for testing.
